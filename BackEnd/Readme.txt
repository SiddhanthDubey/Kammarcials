# REST API CRUD Operations with Python-Flask

# Download and setup

Download the directory containing app.py file, through terminal travel to the directory and run following codes

Step-1: download and activating venv
	
    python -m venv venv
    ./venv/Scripts/activate

Step-2: Installing Dependencies

    pip install mysql-connector-python
    pip install Flask

Step-3: Running application

  $env:PYTHONDONTWRITEBYTECODE=1;$env:FLASK_APP="app";$env:FLASK_ENV = "development"
  flask run
